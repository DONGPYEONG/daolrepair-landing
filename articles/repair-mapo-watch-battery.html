애플워치 배터리 교체 시점 — 자가진단 5가지

  1. 최대 용량 80% 미만 — 설정 → 배터리 → 최대 용량. 80% 미만이면 빠른 교체.
  2. 12시간도 못 가는 빠른 방전 — 정상 사용 시 18~36시간이 정상.
  3. 갑자기 꺼짐 — 배터리 셀 노화로 전류 공급 불안.
  4. 충전 시 발열 — 셀 손상 시작 신호.
  5. 배터리 부풀어 뒷판 들뜸 (위험) — 발화 위험. 즉시 교체 + 사용 중지.
