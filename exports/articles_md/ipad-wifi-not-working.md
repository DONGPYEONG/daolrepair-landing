# 아이패드 와이파이 안됨 — 소프트웨어 설정 vs 안테나 고장 구분법

아이패드 와이파이가 갑자기 안 됩니다. 와이파이 아이콘이 아예 뜨지 않거나, 연결은 되는데 인터넷이 안 되거나, 연결됐다가 바로 끊깁니다. 이런 증상의 원인은 크게 두 가지 — 소프트웨어 설정 문제이거나, 안테나 하드웨어 고장입니다.

대부분은 설정 초기화로 해결됩니다. 하지만 그래도 안 되면 안테나 점검이 필요합니다. 지금 해야 할 순서를 알려드립니다.

## 지금 당장 — 이 순서로 해보세요

1
와이파이 껐다 켜기
설정 → Wi-Fi → 와이파이 토글을 껐다가 10초 후 다시 켜세요. 제어센터의 와이파이 아이콘은 '일시 해제'라서 완전한 끄기가 아닙니다. 반드시 설정 앱에서 해야 합니다.
2
해당 와이파이 네트워크 삭제 후 재연결
설정 → Wi-Fi → 연결된 네트워크 옆 (i) 버튼 → '이 네트워크 지우기' 탭 → 다시 비밀번호 입력하여 연결. 손상된 연결 정보가 초기화됩니다.
3
네트워크 설정 재설정
설정 → 일반 → 전송 또는 iPad 재설정 → 재설정 → 네트워크 설정 재설정. 저장된 모든 와이파이 비밀번호가 삭제되지만 소프트웨어 문제의 약 70%가 이것으로 해결됩니다.
4
다른 와이파이로 테스트
스마트폰 핫스팟이나 다른 장소의 와이파이에 연결해보세요. 다른 와이파이에도 안 되면 아이패드 안테나 문제, 다른 와이파이에는 잘 되면 공유기 문제입니다.

## 원인별 — 소프트웨어 vs 하드웨어 판단

자가 해결 가능
iPadOS 업데이트 후 와이파이 불안정 / 특정 공유기에서만 안 됨
네트워크 설정 재설정으로 대부분 해결됩니다. 공유기의 주파수 대역(2.4GHz/5GHz)을 변경해보는 것도 방법입니다. iPadOS 업데이트 후 문제가 생겼다면 차기 업데이트에서 수정될 수 있습니다.
자가 해결 가능
VPN이나 사설 DNS 설정 충돌
설정 → Wi-Fi → 연결된 네트워크 (i) → DNS 구성이 '수동'으로 되어 있다면 '자동'으로 변경하세요. VPN 앱이 설치되어 있다면 비활성화 후 테스트해보세요.
수리 필요
어떤 와이파이에도 연결 불가 / 와이파이 목록 자체가 안 뜸
설정에서 와이파이 토글이 회색(비활성화)이거나, 와이파이를 켜도 네트워크 목록이 전혀 나타나지 않으면 안테나 케이블 단선 또는 와이파이 칩 고장입니다. 수리점 진단이 필요합니다.
수리 필요
와이파이 신호가 매우 약하게 잡힘 (1칸) / 가까이 가야만 연결됨
안테나 성능이 저하된 상태입니다. 낙하 충격이나 수리 이력이 있는 아이패드에서 자주 발생합니다. 안테나 케이블 교체로 해결되는 경우가 많습니다.

> **이런 상황이라면 즉시 수리점 방문하세요**: 와이파이 토글이 회색으로 비활성화 — 와이파이 칩 또는 안테나 하드웨어 고장입니다. 블루투스도 함께 안 됨 — 와이파이와 블루투스는 같은 칩을 사용합니다. 둘 다 안 되면 칩 레벨 고장입니다. 최근 낙하 이력이 있고 와이파이 신호가 매우 약해짐 — 안테나 케이블이 충격으로 손상되었을 수 있습니다.

> **공유기 문제 확인 팁**: 같은 공유기에 연결된 다른 기기(스마트폰, 노트북 등)도 와이파이가 불안정하다면 공유기 문제일 가능성이 높습니다. 공유기 전원을 30초간 뽑았다가 다시 꽂아보세요. 공유기 펌웨어 업데이트도 확인해보세요.

> **다올리페어에서 무료 진단 받으세요**: 와이파이 문제는 설정인지 안테나인지에 따라 처리 방법이 완전히 달라집니다. 다올리페어에서는 방문 후 무료 진단을 통해 정확한 원인을 파악하고 최적의 해결책을 안내해 드립니다. 진단만 받고 수리를 안 하셔도 됩니다.
