# 다른 곳에서 수리 받다가 오는 분들 — 재수리 사례와 원인

다올리페어에 오시는 분들 중 상당수가 "다른 곳에서 수리 받았는데 또 고장 났어요" 라고 말씀하십니다. 수리 후 며칠 만에 같은 증상이 재발하거나, 없던 문제가 새로 생기기도 합니다.

이 글은 특정 수리점을 비난하려는 것이 아닙니다. 재수리가 발생하는 기술적 원인 을 정리해서, 수리를 맡기기 전에 어떤 점을 확인해야 하는지 알려드리는 것이 목적입니다.

## 재수리 사례 1 — 호환 부품 품질 문제

자주 발생하는 증상
화면 교체 후 터치 오작동, 색감 이상, 밝기 불균일
화면을 교체했는데 며칠 뒤 터치가 튀거나, 색감이 누렇게 변하거나, 화면 한쪽이 어두운 경우입니다. 저가 호환 부품은 외관은 비슷해도 터치 센서 정밀도, 디스플레이 색재현력, 밝기 균일도에서 차이가 큽니다.
호환 부품에도 등급이 있습니다. 최상위 등급(OEM급)은 원래 부품과 거의 동일한 품질을 보여주지만, 최하위 등급은 수리 직후부터 문제가 드러납니다. 문제는 소비자 입장에서 등급을 구별하기 어렵다는 것입니다.

> **확인 포인트**: 수리 전에 "어떤 등급의 부품을 사용하나요?"라고 물어보세요. 부품 등급을 명확히 설명해주는 수리점이 신뢰할 수 있는 곳입니다. 다올리페어는 모든 부품의 등급과 원산지를 수리 전에 안내합니다.

## 재수리 사례 2 — 방수 접착제 미시공

자주 발생하는 증상
화면 교체 후 물 들어감, 습기로 인한 내부 부식
아이폰은 화면과 본체 사이에 방수 접착제가 시공되어 있습니다. 화면을 분리하면 이 접착제가 파손됩니다. 수리 후 새 접착제를 시공하지 않으면 방수 등급이 사실상 사라집니다. 비를 맞거나 세면대 근처에서 사용하다가 내부에 물이 들어가는 사례가 많습니다.
방수 접착제 시공은 별도의 재료와 시간이 필요합니다. 빠른 수리를 강조하는 곳에서는 이 과정을 생략하는 경우가 있습니다. 방수 처리 여부는 수리 전에 반드시 확인하셔야 합니다.

## 재수리 사례 3 — 나사 빠뜨림 또는 잘못 체결

자주 발생하는 증상
수리 후 진동, 딸깍 소리, 화면 들뜸, 내부 쇼트
아이폰 내부에는 모델에 따라 50개 이상의 나사가 있고, 종류와 길이가 제각각입니다. 긴 나사를 짧은 나사 자리에 넣으면 기판을 관통해서 손상시킬 수 있습니다(이를 '롱 스크류 데미지'라고 합니다). 나사를 빠뜨리면 내부 부품이 유동하면서 접촉 불량이나 쇼트가 발생합니다.

> **롱 스크류 데미지는 치명적입니다**: 긴 나사가 기판을 관통하면 메인보드 손상 으로 이어집니다. 이 경우 단순 부품 교체가 아니라 기판 수리가 필요하며, 수리 비용이 크게 올라갑니다. 수리 후 화면에 줄이 생기거나 터치가 부분적으로 안 되면 이 문제를 의심해야 합니다.

## 재수리 사례 4 — Face ID 케이블 미연결

자주 발생하는 증상
화면 교체 후 Face ID 작동 안 함
Face ID 모듈은 메인보드와 개별 페어링되어 있습니다. 화면 교체 시 기존 화면의 Face ID 케이블을 새 화면으로 옮겨야 합니다. 이 과정을 생략하거나 케이블을 손상시키면 Face ID가 영구적으로 작동하지 않습니다. iOS 16 이후 부품 페어링 정책이 더 엄격해져서, 한번 끊어진 연결은 복구가 매우 어렵습니다.
Face ID 외에도 트루톤(True Tone) 디스플레이 기능 이 사라지는 경우가 있습니다. 이 역시 화면 교체 시 기존 데이터를 새 부품으로 이전하지 않으면 발생합니다.

## 재수리를 피하려면 — 수리 전 확인 체크리스트

1
부품 등급 확인
"어떤 등급의 부품을 사용하나요?" 부품 등급을 명확히 설명해주는지 확인하세요. 가격만 말하고 등급을 설명 안 하면 주의가 필요합니다.
2
방수 처리 여부 확인
"수리 후 방수 접착제를 다시 시공하나요?" 방수 처리를 안 하면 수리 후 물 들어갈 위험이 있습니다.
3
Face ID / True Tone 유지 여부 확인
"수리 후 Face ID랑 True Tone 유지되나요?" 화면 교체 시 반드시 확인해야 할 항목입니다.
4
A/S 보증 기간 확인
"수리 후 A/S 기간은 얼마나 되나요?" 보증 기간이 없거나 짧으면 재수리 시 추가 비용이 발생합니다.

> **다올리페어는 재수리도 책임집니다**: 다올리페어에서는 다른 곳에서 수리 받은 기기도 무료 진단 후 정확한 원인을 파악해 드립니다. 수리 후 3개월 무상 A/S를 제공하며, 부품 등급과 작업 내역을 투명하게 안내합니다. 진단만 받고 수리를 안 하셔도 됩니다.
