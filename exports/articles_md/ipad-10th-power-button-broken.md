아이패드 10세대 전원 버튼을 누르는데 반응이 없습니다. 또는 버튼이 안으로 들어가서 눌리지 않습니다. 물리 버튼 함몰 또는 내부 접촉불량 증상입니다.

전원 버튼은 매일 수십 번 누르는 부품입니다. 낙하 충격이 아니더라도 오래 사용하면 내부 금속 돔 스위치가 약해지거나, 주변 프레임이 미세하게 변형되면서 접촉불량이 발생합니다. 지금 할 수 있는 것과 수리가 필요한 상황을 구분해 드립니다.

## 지금 당장 — 이 순서로 해보세요

1
AssistiveTouch 켜기 (임시 해결)
설정 → 손쉬운 사용 → 터치 → AssistiveTouch → 켜기. 화면에 가상 버튼이 생깁니다. 기기 → 화면 잠금으로 전원 버튼 없이도 잠금이 가능합니다. 당장 사용이 급할 때 임시 방편입니다.
2
버튼 주변 이물질 확인
버튼 틈새에 먼지나 이물질이 끼어 눌림을 방해할 수 있습니다. 부드러운 솔이나 에어 더스터로 버튼 주변을 청소해보세요. 간혹 이것만으로 해결되기도 합니다.
3
iCloud 백업 실행
버튼이 완전히 안 되기 전에 백업하세요. 설정 → [이름] → iCloud → iCloud 백업 → 지금 백업. 전원 버튼이 완전히 고장 나면 강제 재시작도 못 합니다.
4
수리점 진단 받기
버튼 함몰이나 반복적인 접촉불량은 자가 수리가 어렵습니다. 내부 케이블이나 돔 스위치 교체가 필요하므로 전문 수리점을 방문하세요.

## 고장 유형별 — 원인과 수리 필요성 판단

🟢 이물질로 인한 눌림 방해
버튼이 뻑뻑하거나 가끔 안 눌림
버튼 틈새에 먼지나 이물질이 끼어 있습니다. 에어 더스터로 청소하면 해결될 수 있습니다. 청소 후에도 반복되면 내부 문제입니다.
🔴 내부 돔 스위치 손상
버튼을 눌러도 클릭감이 없음 / 반응 없음
버튼 내부의 금속 돔 스위치가 변형되거나 닳아서 접점이 닿지 않습니다. 돔 스위치 또는 버튼 케이블 교체 수리가 필요합니다.
🔴 버튼 함몰 (프레임 변형)
버튼이 안으로 들어가서 눌리지 않음
낙하나 외부 압력으로 버튼 주변 프레임이 미세하게 변형되어 버튼이 안으로 밀려 들어간 상태입니다. 프레임 교정과 버튼 부품 교체가 필요합니다.

> **AssistiveTouch로 당장 쓸 수 있습니다**: 전원 버튼이 안 돼도 AssistiveTouch를 켜면 화면 잠금, 스크린샷, 볼륨 조절까지 가상 버튼으로 가능합니다. 수리 전까지 임시로 사용하기에 충분합니다. 하지만 강제 재시작이 필요한 상황이 오면 물리 버튼 없이는 대처가 어려우므로, 가능한 빨리 수리하시는 것을 권장합니다.

> **이런 상황이라면 빠르게 수리하세요**: 버튼이 완전히 안 눌린다면 — 강제 재시작이 불가능해져서, 소프트웨어 문제가 생겼을 때 대처가 안 됩니다. 버튼이 가끔 되다가 안 되기를 반복한다면 — 접촉불량은 시간이 지나면 완전 고장으로 이어집니다. 배터리 최대 용량이 80% 미만이라면 — 배터리 노화와 버튼 문제가 겹치면 갑자기 꺼진 후 다시 켜지지 않을 수 있습니다.

> **다올리페어에서 무료 진단 받으세요**: 전원 버튼 수리는 내부를 열어 원인을 정확히 파악해야 합니다. 돔 스위치 문제인지, 케이블 문제인지, 프레임 변형인지에 따라 수리 방법과 비용이 달라집니다. 다올리페어에서 무료 진단 후 가장 합리적인 수리 방법을 안내해 드립니다.
