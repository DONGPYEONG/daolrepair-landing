## Face ID 안될 때 원인 3가지

Face ID 오류는 크게 세 가지 원인으로 나뉩니다. 수리 현장에서 보면 상당수가 소프트웨어 문제로 재설정만으로 해결됩니다. 하지만 낙하 이후 발생하거나 화면 수리 후 발생했다면 하드웨어를 의심해야 합니다. 증상에 따라 대응 방법이 완전히 다르니 먼저 원인을 파악하는 게 중요합니다.

## 소프트웨어 해결 먼저 시도하기

하드웨어 고장이 아닌 경우, 아래 방법을 순서대로 시도하면 대부분 해결됩니다. 특히 iOS를 업데이트한 직후에 Face ID가 작동하지 않는 경우는 재시동만으로도 해결되는 경우가 많습니다.

1단계 · 아이폰 재시동
전원을 완전히 끄고 다시 켜보세요. 소프트웨어 오류로 Face ID가 일시적으로 비활성화된 경우 재시동만으로 해결됩니다.
2단계 · Face ID 데이터 재등록
설정 → Face ID 및 암호 → 'Face ID 재설정'을 누른 뒤, 얼굴 데이터를 다시 등록합니다. 시간이 지나면서 얼굴이 변했거나 안경을 새로 맞춘 경우에 효과적입니다.
3단계 · iOS 업데이트 확인
설정 → 일반 → 소프트웨어 업데이트. iOS 버그로 Face ID가 오작동하는 사례가 있으며, 업데이트로 해결되는 경우가 있습니다.
4단계 · 카메라 전면부 이물질 확인
트루뎁스 카메라가 있는 전면 상단(노치 또는 다이나믹 아일랜드 영역)에 보호 필름이나 케이스가 카메라를 가리고 있지 않은지 확인하세요.

> **💡 Face ID가 잠긴 경우**: 5회 연속 인증 실패 시 Face ID가 잠깁니다. 이 경우 비밀번호를 한 번 입력하면 잠금이 해제되고 다시 Face ID를 사용할 수 있습니다. 비밀번호를 잊어버린 경우에는 아이폰을 초기화해야 합니다.

## 하드웨어 고장인 경우 — 공식 센터 vs 사설 수리점

소프트웨어 해결 방법을 모두 시도했는데도 Face ID 메뉴 자체가 사라지거나 "Face ID를 사용할 수 없습니다"가 계속 뜬다면 하드웨어 고장입니다. 이 경우 공식 센터와 사설 수리점의 차이를 반드시 알고 결정해야 합니다.

애플 공식 센터
- Face ID 완전 복구 가능
- 트루뎁스 모듈과 메인보드 재연동 가능
- 비용이 높고 (20~60만원 이상) 대기 시간 길 수 있음
- AppleCare+ 가입 시 수리비 절감

사설 수리점
- 셀피(전면) 카메라 교체는 가능
- Face ID 트루뎁스 모듈 교체 시 Face ID 비활성화
- 비용은 저렴하고 당일 처리 가능
- Face ID 복구는 불가능

> **⚠ 반드시 알아야 할 중요한 사실**: Face ID의 트루뎁스 카메라 모듈은 아이폰 메인보드와 암호화 방식으로 1:1 쌍을 이뤄 연동됩니다. 사설 수리점에서 이 모듈을 교체하면 기기와의 암호화 연동이 끊기기 때문에 Face ID가 영구적으로 비활성화됩니다. 단, 셀피(전면) 카메라 단독 고장은 Face ID와 별개이므로 사설에서도 교체 가능합니다.

## Face ID 고장 예방법

Face ID는 전면 상단의 트루뎁스 카메라 시스템이 작동하는 기능입니다. 이 부위가 가장 취약한 상황은 화면 파손 및 낙하 충격입니다. 예방을 위한 몇 가지 실천 사항을 안내합니다.

1
— 전면 카메라 영역을 가리거나 누르는 보호 필름은 Face ID 인식에 영향을 줄 수 있습니다.
2
— 화면이 깨진 채로 방치하면 내부 모듈까지 손상이 번질 수 있습니다. 특히 상단 노치나 다이나믹 아일랜드 주변 파손은 Face ID에 직결됩니다.
3
— 화면 교체 수리 중 실수로 트루뎁스 모듈 커넥터가 손상될 수 있습니다. 수리 후 Face ID가 작동하는지 현장에서 바로 확인하세요.
4
— Face ID 하드웨어 수리는 공식 센터만 가능하고 비용이 높습니다. Face ID가 중요하다면 AppleCare+ 가입이 장기적으로 유리합니다.
