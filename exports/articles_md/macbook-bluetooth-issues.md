# 맥북 블루투스 끊김 — 에어팟·마우스·키보드 연결 불안정 해결법

> 맥북 블루투스가 자꾸 끊깁니다. 에어팟, 매직 마우스, 키보드 연결이 불안정할 때 소프트웨어 리셋부터 블루투스 모듈 고장까지 단계별 해결법을 정리합니다 · 90일 보증 · 실패 시 비용 0원

**원문**: [https://xn--2j1bq2k97kxnah86c.com/articles/macbook-bluetooth-issues.html](https://xn--2j1bq2k97kxnah86c.com/articles/macbook-bluetooth-issues.html)

---

맥북에서 에어팟이 자꾸 끊기거나, 매직 마우스가 움직이다 멈추거나, 블루투스 키보드 입력이 안 됩니다. 블루투스 문제의 90%는 소프트웨어 리셋으로 해결됩니다. 순서대로 해보세요.

## 단계별 해결 — 이 순서로 해보세요

1
블루투스 끄고 다시 켜기
시스템 설정 → 블루투스 → 끄기 → 10초 후 다시 켜기. 가장 간단하지만 의외로 효과적입니다.
2
기기 연결 해제 후 다시 페어링
블루투스 설정에서 문제 기기 옆의 'X' 또는 '이 기기 지우기'를 클릭하고, 기기를 페어링 모드로 전환한 뒤 다시 연결하세요.
3
블루투스 모듈 리셋
Shift+Option을 누른 상태에서 메뉴바의 블루투스 아이콘을 클릭 → '블루투스 모듈 재설정' 선택. 모든 블루투스 연결이 초기화되고 다시 시작됩니다.
4
블루투스 plist 파일 삭제
Finder → 이동 → 폴더로 이동 → /Library/Preferences 입력 → com.apple.Bluetooth.plist 파일을 삭제 → 재시작. 블루투스 설정이 완전히 초기화됩니다.
5
NVRAM/SMC 초기화
Intel: NVRAM(Option+Cmd+P+R 20초) + SMC 초기화. Apple Silicon: 전원 끄고 30초 기다렸다 재시작. 하드웨어 수준의 블루투스 설정이 초기화됩니다.

> **특정 기기만 끊길 때**: 에어팟만 끊기고 마우스는 괜찮다면, 에어팟 자체 문제일 수 있습니다. 해당 기기를 다른 기기(아이폰 등)에 연결해서 테스트 해보세요. 다른 기기에서도 끊기면 에어팟 문제, 맥북에서만 끊기면 맥북 블루투스 문제입니다.

## 이건 하드웨어 고장입니다 — 수리가 필요한 경우

수리 필요
위의 모든 방법을 해도 블루투스가 아예 켜지지 않음
블루투스 아이콘에 물결 표시가 나오거나 '사용 불가'로 표시된다면 블루투스/Wi-Fi 통합 모듈의 하드웨어 고장입니다.
수리 필요
연결은 되지만 1m 이내에서도 끊김이 반복
블루투스 안테나 케이블이 손상됐거나 모듈 자체의 송수신 성능이 떨어진 상태입니다. 침수 이력이 있는 맥북에서 자주 발생합니다.
참고
USB 3.0 허브 사용 시 간헐적 끊김
USB 3.0 장치가 2.4GHz 주파수에 간섭을 일으켜 블루투스 연결이 불안정해질 수 있습니다. USB 허브를 빼고 테스트해보세요. 이 경우 하드웨어 고장이 아닙니다.

> **수리비 안내 (2026년 4월 기준)**: 블루투스/Wi-Fi 모듈 교체 — 사설 수리점 기준 10~20만 원. 맥북의 블루투스와 Wi-Fi는 하나의 모듈에 통합되어 있습니다. Apple Silicon 모델 주의 — M1 이후 모델은 블루투스 칩이 메인보드에 통합된 경우가 있어, 모듈만 교체가 안 될 수 있습니다. 정확한 진단이 필요합니다.

> **다올리페어에서 블루투스 문제 진단받으세요**: 소프트웨어 리셋으로도 해결이 안 된다면 무료 진단을 받아보세요. 모듈 고장인지, 안테나 문제인지, 메인보드 문제인지 정확히 판단해 드립니다. 진단만 받고 수리를 안 하셔도 됩니다.
