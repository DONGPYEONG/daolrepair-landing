# 사설 수리 후 애플 공식 서비스 거부당하는 경우

사설 수리점에서 화면을 교체했는데, 나중에 다른 문제로 애플 서비스센터에 갔더니 "승인되지 않은 수정이 감지되었습니다" 라는 안내를 받았다는 사례가 종종 있습니다. 정확히 어떤 일이 일어나는지 알아보겠습니다.

## 애플이 공식 서비스를 거부하는 사유

거부 사유 1: 비순정 부품 감지
화면, 배터리, 카메라 등에 호환 부품 사용
애플은 부품의 시리얼 번호를 확인합니다. 순정 부품이 아닌 호환 부품이 감지되면 해당 부분의 수리를 거부합니다. iOS 설정 → 일반 → 정보에서 "알 수 없는 부품" 경고가 표시되며, 이는 애플 진단 프로그램에서도 확인됩니다.
거부 사유 2: 사설 수리로 인한 추가 손상
내부 케이블 손상, 나사 누락, 방수 테이프 훼손
기술력이 부족한 수리점에서 수리하면 내부에 추가 손상이 생길 수 있습니다. 케이블 핀 손상, 나사 누락, 커넥터 파손 등이 발견되면 "승인되지 않은 수정으로 인한 손상"으로 판단하여 전체 수리를 거부할 수 있습니다.
거부 사유 3: 방수 씰(seal) 훼손
분해 과정에서 방수 테이프가 제거됨
아이폰을 분해하면 방수 씰이 훼손됩니다. 공식 수리에서는 방수 씰을 다시 부착하지만, 사설 수리에서는 하지 않는 경우가 많습니다. 이후 침수가 발생하면 "사설 수리로 인한 방수 기능 상실"로 판단하여 수리를 거부합니다.

## 영향 범위 — 어디까지 거부되나?

핵심은 이것입니다: 사설 수리한 부분만 거부되는 게 원칙이지만, 추가 손상이 있으면 전체가 거부될 수 있습니다.

1
부분 거부 — 수리한 해당 부분만
화면을 사설 수리했다면, 화면 관련 수리만 거부됩니다. 배터리 교체, 충전 포트 수리 등 다른 부분은 공식 서비스를 받을 수 있습니다. 단, 사설 수리가 깔끔하게 되어 추가 손상이 없어야 합니다.
2
전체 거부 — 추가 손상이 발견된 경우
사설 수리 과정에서 내부 커넥터 손상, 나사 누락 등이 발견되면 기기 전체의 공식 수리가 거부될 수 있습니다. "승인되지 않은 수정으로 인한 기기 손상"이 사유입니다.
3
기능 제한 — 트루톤, Face ID 등
사설 화면 수리 시 트루톤 데이터를 이전하지 않으면 트루톤 기능이 비활성화됩니다. Face ID 모듈을 교체하면 Face ID가 작동하지 않을 수 있습니다. 이는 거부라기보다 기능 제한에 해당합니다.

> **특히 주의: Face ID와 배터리 경고**: Face ID — 사설 수리 시 Face ID 모듈을 분리하면 Face ID가 영구적으로 비활성화될 수 있습니다. 화면 수리 시 Face ID 모듈을 건드리지 않는 수리점을 선택하세요. 배터리 경고 — 비순정 배터리로 교체하면 설정에서 "서비스" 경고가 표시되고, 배터리 상태 정보를 확인할 수 없게 됩니다. 기능에는 영향이 없지만 중고 판매 시 감가 요인이 됩니다.

## 사설 수리 전 확인 체크리스트

1
애플케어+ / 보증 상태 먼저 확인
설정 → 일반 → 정보 → 보증 상태. 애플케어+가 활성 상태이거나 보증 기간 내라면 공식 수리가 거의 항상 유리합니다. 사설 수리를 받으면 보증 혜택을 잃게 됩니다.
2
수리점의 트루톤/Face ID 이전 가능 여부
화면 수리 시 트루톤 데이터 이전이 가능한지 확인하세요. 전용 장비가 있는 수리점이 좋습니다. Face ID 모듈은 건드리지 않고 수리하는지도 확인하세요.
3
수리점 A/S 보증 정책
사설 수리 후 동일 부분에 문제가 생기면 어떻게 처리하는지 확인하세요. 3개월 이상 무상 A/S를 보장하는 수리점이 신뢰할 만합니다.
4
향후 공식 서비스 필요 여부 판단
앞으로 공식 서비스를 받을 가능성이 높다면(애플케어+ 보유, 보증 기간 내) 사설 수리를 피하세요. 보증이 만료되고 애플케어+도 없다면, 공식 서비스 거부의 실질적 영향은 크지 않습니다.

> **결론: 사설 수리, 이럴 때 하세요**: 보증이 만료되었고 , 애플케어+가 없으며 , 공식 수리비가 부담된다면 사설 수리는 합리적인 선택입니다. 단, 트루톤 이전이 가능하고 A/S 보증이 있는 신뢰할 수 있는 수리점을 선택하세요. 보증이 남아있다면 공식 수리를 권장합니다.

> **다올리페어 — 트루톤 이전 + 3개월 A/S**: 다올리페어는 화면 교체 시 트루톤 데이터 이전을 기본으로 진행하며, Face ID 모듈을 건드리지 않습니다. 수리 후 3개월 무상 A/S를 보장하고, 수리 전 공식 서비스 영향도 사전에 안내해 드립니다. 걱정되시면 먼저 무료 상담을 받으세요.
