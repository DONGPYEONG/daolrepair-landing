# 애플워치 초기화 방법 — 판매 전·수리 전 완전 가이드

> 애플워치를 판매하거나 수리 맡기기 전, 반드시 해야 하는 초기화와 페어링 해제 방법을 단계별로 안내합니다. 활성화 잠금 해제까지 포함한 완전 가이드입니다 · 90일 보증 · 실패 시 비용 0원

애플워치를 중고로 판매하거나 수리를 맡기기 전에는 반드시 초기화를 해야 합니다. 초기화를 안 하면 내 개인 정보가 남아 있고, 활성화 잠금 때문에 다른 사람이 사용할 수 없게 됩니다.

특히 수리를 맡길 때 활성화 잠금이 걸려 있으면 수리 후 테스트가 안 되어 수리 자체가 지연될 수 있습니다. 아래 순서대로 하시면 빠짐없이 처리됩니다.

## 방법 1 — 아이폰에서 페어링 해제 (권장)

이 방법이 가장 안전합니다. 페어링을 해제하면 자동으로 백업이 생성 되고, 활성화 잠금도 함께 해제됩니다.

1
아이폰과 애플워치를 가까이 두기
페어링 해제를 위해서는 두 기기가 블루투스 범위 안에 있어야 합니다. 나란히 놓으세요.
2
아이폰에서 Watch 앱 열기
아이폰 홈 화면에서 'Watch' 앱(시계 모양 아이콘)을 탭합니다.
3
'나의 시계' 탭 > 상단의 내 워치 선택
화면 상단에 내 애플워치 이름이 보입니다. 탭하세요.
4
'i' 아이콘 탭 > 'Apple Watch 연결 해제' 탭
워치 이름 옆의 정보(i) 버튼을 누르고, 'Apple Watch 연결 해제'를 선택합니다. 확인 메시지가 뜨면 다시 한 번 '연결 해제'를 탭합니다.
5
Apple ID 비밀번호 입력
활성화 잠금을 해제하기 위해 Apple ID 비밀번호를 입력합니다. 이 과정이 완료되면 애플워치가 자동으로 초기화됩니다.

> **이 방법의 장점**: 백업이 자동 생성됩니다. 나중에 같은 아이폰이나 새 아이폰으로 다시 페어링하면 '백업에서 복원'을 선택해 이전 데이터를 되찾을 수 있습니다. 활성화 잠금도 자동 해제됩니다.

## 방법 2 — 애플워치에서 직접 초기화

아이폰이 없거나 연결이 안 되는 경우 이 방법을 사용합니다. 다만 이 방법으로는 백업이 생성되지 않고, 활성화 잠금이 해제되지 않을 수 있습니다.

1
설정 앱 열기
애플워치에서 설정(톱니바퀴) 앱을 엽니다.
2
일반 > 재설정
일반을 탭하고, 맨 아래로 스크롤해서 '재설정'을 탭합니다.
3
'모든 콘텐츠 및 설정 지우기' 선택
비밀번호 입력 후 확인하면 초기화가 시작됩니다. 완료까지 몇 분 소요됩니다.

> **활성화 잠금 주의**: 방법 2로 초기화하면 활성화 잠금이 남아 있을 수 있습니다. 판매 목적이라면 반드시 방법 1(아이폰에서 페어링 해제)을 사용하세요. 활성화 잠금이 남아 있으면 구매자가 애플워치를 설정할 수 없습니다. 수리 목적이라면 수리점에서 안내하는 대로 처리하시면 됩니다. 다올리페어에서는 접수 시 필요한 절차를 모두 안내해 드립니다.

## 상황별 초기화 체크리스트

판매 전
중고 판매 시 반드시 해야 할 것
1. 아이폰에서 페어링 해제 (방법 1) → 2. 활성화 잠금 해제 확인 → 3. SIM 제거(셀룰러 모델) → 4. 밴드 분리 후 정리. 이 순서를 빠짐없이 완료해야 구매자가 문제없이 사용할 수 있습니다.
수리 전
수리 맡기기 전 해야 할 것
1. 데이터 백업 확인 (아이폰 Watch 앱에서) → 2. 수리점 안내에 따라 초기화 여부 결정 → 3. Apple ID 비밀번호 준비. 화면 교체/배터리 교체는 초기화 없이도 가능한 경우가 많습니다.
비밀번호 분실
비밀번호를 잊어서 초기화가 안 될 때
애플워치 비밀번호를 잊었다면 아이폰 Watch 앱에서 페어링 해제로 초기화할 수 있습니다. 아이폰도 없다면 애플워치를 충전기에 올려놓고 측면 버튼을 길게 눌러 '전원 끄기' 화면이 나오면, Digital Crown을 길게 눌러 '모든 콘텐츠 및 설정 지우기'를 선택하세요.

> **iCloud에서 원격 초기화하는 방법**: 애플워치가 없거나 고장 상태라면 iCloud.com > 나의 기기 찾기에서 원격으로 지울 수 있습니다. 원격 초기화 후 '계정에서 제거'까지 해야 활성화 잠금이 완전히 해제됩니다.
