# 중고 아이폰 샀다가 후회하는 사람들이 빠뜨린 것들

> 중고 아이폰 구매 전 현장에서 5분이면 확인할 수 있는 7가지 체크리스트. iCloud 잠금, 배터리 성능, 침수 흔적, 수리 이력, IMEI 조회까지 — 후회 없는 중고 구매를 위한 완전 가이드 · 90일 보증 · 실패 시 비용 0원

중고 아이폰 시장은 크게 세 가지로 나뉩니다. 정상적인 기기를 정직하게 파는 사람, 모르고 파는 사람, 그리고 알면서 숨기는 사람. 문제는 사는 입장에서 이 세 가지를 구분하기 어렵다는 겁니다.

수리점을 운영하다 보면 중고 구매 후 점검을 받으러 오는 분들이 정말 많습니다. 배터리가 이상하다, 침수된 것 같다, 잠금이 풀리지 않는다 — 대부분 현장에서 5분만 확인했어도 걸렸을 것들 입니다.

아래 7가지를 순서대로 확인하세요. 어렵지 않습니다. 판매자 앞에서 바로 할 수 있습니다.

## 현장 5분 체크리스트

1. iCloud 잠금
1
iCloud·나의 iPhone 찾기 잠금 해제 여부
가장 중요
중고 아이폰 사기의 90%는 여기서 시작됩니다. iCloud 잠금이 걸려 있으면 기기를 켜도 아무것도 할 수 없습니다. 정품 판매 채널이든 중고 앱이든 관계없이 반드시 직접 확인해야 합니다.

확인 방법
1. 기기를 켜서 잠금 화면을 해제한다
2. 설정 앱 최상단 에 이름과 Apple ID가 보여야 정상
3. "활성화 잠금" 화면이 뜨거나 Apple ID 입력을 요구하면 구매 금지
4. 판매자가 직접 설정 → [이름] → 로그아웃을 눌러 Apple ID를 해제해야 안전

Apple ID 정상 로그인 + 직접 로그아웃 확인
활성화 잠금 화면 → 절대 구매 금지

> **⚠️ 이런 말은 믿지 마세요**: "집에 가면 원격으로 잠금 해제해드릴게요" — 그런 방법은 없습니다. 현장에서 Apple ID 로그아웃이 안 된다면 사지 마세요.

2. 배터리 성능
2
배터리 성능 확인
가장 중요
배터리는 중고 아이폰에서 가장 눈에 띄게 열화되는 부품입니다. 겉보기에 멀쩡해도 배터리가 70%대라면 구매 즉시 교체 비용이 추가됩니다. 반드시 현장에서 숫자로 확인하세요.

확인 방법
1. 설정 → 배터리 → 배터리 상태 및 충전
2. "최대 용량" 숫자를 확인
3. 숫자가 표시되지 않거나 "서비스 필요" 메시지가 뜨면 이상 있음

85% 이상 — 양호
80~84% — 가격 협상 활용
79% 이하 — 교체 비용 별도 발생

> **💡 협상에 활용하세요**: 배터리가 80% 미만이면 교체 비용만큼 가격을 낮춰달라고 협상하거나, 교체 후 파는 곳에서 구입하는 게 낫습니다.

3. 수리 이력
3
수리 이력 · 비정품 부품 사용 여부
중요
아이폰은 비정품 부품이 사용되면 설정에서 경고 메시지를 표시합니다. 판매자가 숨겼더라도 기기가 알려줍니다. 특히 액정과 배터리를 비정품으로 교체한 경우가 많습니다.

확인 방법
1. 설정 → 일반 → 정보
2. 아래로 스크롤 → "부품 및 수리 내역" 항목 확인
3. 각 항목에 "정품 Apple 부품" 또는 경고 여부 표시됨

정품 Apple 부품 표시
알 수 없는 부품 표시 — 비정품 교체 이력
비정품 부품 자체가 사기는 아닙니다. 하지만 그 사실을 판매자가 말하지 않았다면 신뢰도가 낮은 거래입니다. 가격에 반영해서 협상하세요.

4. 침수 흔적
4
침수 흔적 확인 (LCI)
중요
침수된 아이폰은 당장은 멀쩡해 보여도 시간이 지나면서 증상이 나타납니다. 애플은 이를 대비해 기기 내부에 '액침 표시기(LCI, Liquid Contact Indicator)'를 넣어뒀습니다. 물에 닿으면 색이 변합니다.

확인 방법
1. SIM 카드 트레이를 핀으로 꺼낸다
2. 트레이가 들어가는 구멍 안쪽을 라이트로 비춘다
3. 작은 표시기의 색을 확인

흰색 또는 은색 — 정상
빨간색 또는 분홍색 — 침수 이력 있음

> **⚠️ 침수 이력은 숨겨집니다**: 외관이 깨끗해도 침수된 기기가 많습니다. 겨울철 눈비, 세면대 근처, 음료 등 — 판매자도 모르는 경우가 있습니다. 반드시 직접 확인하세요.

5. 통신사 잠금
5
통신사 잠금(SIM Lock) 여부
중요
특정 통신사에 잠긴 기기는 다른 통신사 유심을 넣어도 작동하지 않습니다. 특히 해외 직구폰이나 통신사 약정폰은 잠금이 걸려 있는 경우가 있습니다. 내 유심을 직접 넣어보는 게 가장 확실합니다.

확인 방법
1. 내 유심 카드를 기기에 직접 넣어본다
2. 설정 → 일반 → 정보 → "통신사 잠금" 항목 확인
3. "SIM 잠금 없음" 표시면 안전

SIM 잠금 없음 표시
다른 유심 인식 불가 → 가격 협상 또는 포기
6. IMEI 조회
6
IMEI 조회 — 분실·도난 여부
중요
분실·도난 신고된 기기는 통신 서비스가 정지될 수 있습니다. 겉보기엔 완전히 정상이지만 통신사 조회 시 문제가 나타납니다. IMEI 번호로 미리 확인하세요.

확인 방법
1. 설정 → 일반 → 정보 → IMEI 번호 메모
2. 또는 전화 앱에서 *#06# 입력하면 바로 표시됨
3. IMEI.kr에서 분실·도난 조회 바로가기 →
4. 통신사 고객센터에 전화해서 분실 신고 여부 문의 가능

> **💡 판매자가 IMEI 확인을 거부하면**: 정직한 판매자라면 IMEI 확인을 당연히 허용합니다. 거부하거나 핑계를 댄다면 구매를 재고하세요.

## 이런 판매자라면 조심하세요

기기 상태와 별개로, 판매자의 태도에서도 위험 신호가 보입니다.

🚩
현장 확인을 거부하거나 서두르는 판매자는 뭔가 숨기고 있을 가능성이 높습니다.
🚩
"배터리는 좋아요"라고 말만 하고 직접 보여주지 않는다면 신뢰하기 어렵습니다.
🚩
Apple ID 로그아웃은 반드시 현장에서, 판매자가 직접 해야 합니다.
🚩
정상 기기를 이유 없이 훨씬 싸게 파는 사람은 거의 없습니다. 가격이 너무 싸면 이유가 있습니다.
🚩
직접 확인할 수 없는 거래 방식은 사기의 전형적인 패턴입니다.

> **실제로 접수된 상황들**: "배터리 성능이 79%인데 판매자가 '멀쩡하다'고 했어요" "iCloud 잠금이 걸려있는데 사버렸어요" "침수된 건지 몰랐는데 지금 계속 이상해요" "비정품 액정인지 모르고 샀는데 색이 이상해요"
