# 맥북 외부 모니터 연결 안 됨 — 원인별 점검 순서

맥북에 모니터를 연결했습니다. 그런데 모니터에 아무것도 안 나옵니다. 또는 연결은 됐는데 화면이 깜빡이거나, 해상도가 이상합니다. 외부 모니터 문제는 원인이 다양하지만, 순서대로 점검하면 대부분 해결됩니다.

## 지금 당장 — 이 순서로 점검하세요

1
케이블을 뽑았다 다시 꽂기
가장 먼저 할 일입니다. 케이블 양쪽(맥북 쪽, 모니터 쪽) 모두 완전히 뽑았다가 단단히 꽂아보세요. USB-C 케이블은 살짝 빠져있어도 안 될 수 있습니다. 단순 접촉 불량인 경우가 의외로 많습니다.
2
다른 포트로 연결 시도
맥북에 USB-C 포트가 여러 개 있다면 다른 포트에 꽂아보세요. 특정 포트만 문제가 있을 수 있습니다. HDMI 포트가 있는 맥북 프로라면 HDMI로도 시도해보세요.
3
허브 없이 직접 연결
USB-C 허브나 독을 통해 연결하고 있다면, 허브를 빼고 맥북에서 모니터로 직접 연결해보세요. 허브의 영상 출력 호환성 문제인 경우가 많습니다.
4
디스플레이 설정 확인
시스템 설정 → 디스플레이에서 외부 모니터가 감지되는지 확인하세요. 감지는 됐는데 화면이 안 보인다면 '미러링' 또는 '확장' 설정을 변경해보세요. Option 키를 누른 채 '디스플레이 감지' 버튼을 클릭하면 강제 감지가 됩니다.
5
맥북 재시작
모니터를 연결한 상태에서 맥북을 재시작해보세요. macOS의 디스플레이 드라이버가 새로 초기화되면서 인식되는 경우가 있습니다.

## 원인별 판단 — 소프트웨어 vs 하드웨어

직접 해결 가능
케이블 불량 또는 호환성 문제
다른 케이블로 바꿨더니 되는 경우입니다. 특히 USB-C to HDMI 케이블은 품질 차이가 큽니다. 4K 60Hz를 원한다면 HDMI 2.0 이상 인증 케이블이 필요하고, Thunderbolt 모니터라면 Thunderbolt 케이블이 필요합니다.
직접 해결 가능
허브/독 호환성 문제
허브를 빼고 직접 연결하면 되는 경우입니다. 저가형 USB-C 허브는 DP Alt Mode를 제대로 지원하지 않거나, 전력이 부족하여 영상 출력이 불안정할 수 있습니다. Thunderbolt 인증 독을 사용하면 대부분 해결됩니다.
직접 해결 가능
지원 대수 초과
M1/M2 맥북 에어는 외부 모니터 1대만 지원합니다. M3 에어는 클램셸 모드에서 최대 2대. 맥북 프로는 칩에 따라 2~4대. 지원 대수를 초과한 모니터는 인식되지 않습니다. Apple 메뉴 → 이 Mac에 관하여에서 모델과 지원 사양을 확인하세요.
수리 필요
USB-C 포트 물리적 손상
포트 안쪽 핀이 휘거나, 이물질이 끼어 접촉이 안 되는 경우입니다. 다른 포트에서는 정상이고 특정 포트에서만 안 된다면 포트 수리가 필요합니다.
수리 필요
GPU 또는 메인보드 문제
모든 포트, 모든 케이블, 모든 모니터에서 외부 출력이 안 된다면 GPU의 외부 출력 회로에 문제가 있을 수 있습니다. 맥북 자체 화면은 정상인데 외부 출력만 안 되는 것이 특징입니다. 메인보드 수리가 필요합니다.

> **이런 증상이면 수리점 방문을 권장합니다**: 모든 케이블과 포트에서 안 됨 — 케이블 3개 이상, 포트 2개 이상 시도했는데 전부 안 되면 GPU 문제를 의심하세요. 연결 시 맥북이 멈추거나 재시작됨 — 모니터를 꽂는 순간 맥북이 프리즈되거나 꺼지면 메인보드 쇼트 가능성이 있습니다. 즉시 사용을 중단하세요. 화면에 줄이 가거나 색상이 심하게 깨짐 — 단순 해상도 문제가 아니라 GPU 하드웨어 이상일 수 있습니다.

## 모니터 연결 팁

> **안정적인 외부 모니터 연결을 위한 팁**: 케이블은 인증 제품을 사용하세요. USB-C to HDMI는 HDMI 2.0 이상 인증, Thunderbolt 연결은 Thunderbolt 인증 케이블을 사용하면 호환성 문제가 크게 줄어듭니다. 허브보다 독(Dock)을 추천합니다. 저가 허브는 영상 출력이 불안정합니다. 업무용이라면 Thunderbolt 독에 투자하는 것이 장기적으로 안정적입니다. macOS 업데이트를 유지하세요. Apple은 macOS 업데이트를 통해 디스플레이 호환성을 지속적으로 개선합니다. 최신 버전이 아니라면 업데이트 후 다시 시도해보세요.

> **다올리페어에서 무료 진단 받으세요**: 케이블과 설정을 다 확인했는데도 안 된다면 GPU나 포트 하드웨어 점검이 필요합니다. 다올리페어에서는 방문 후 무료 진단을 통해 정확한 원인을 파악하고 최적의 해결책을 안내합니다. 진단만 받고 수리를 안 하셔도 됩니다.
