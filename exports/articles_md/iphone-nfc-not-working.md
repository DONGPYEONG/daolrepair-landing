편의점에서 애플페이로 결제하려는데 반응이 없습니다. 지하철 교통카드도 안 됩니다. 아이폰 NFC 기능이 작동하지 않는 상황입니다.

NFC(Near Field Communication)는 아이폰 상단에 내장된 근거리 통신 기능입니다. 애플페이, 교통카드, NFC 태그 읽기 등에 사용됩니다. 이 기능이 갑자기 안 될 때, 소프트웨어 문제인지 하드웨어 고장인지 구분하는 방법을 알려드립니다.

## 자가 점검 — 이 순서대로 확인하세요

1
아이폰 상단을 단말기에 대고 있는지 확인
아이폰 NFC 안테나는 상단 카메라 부근에 있습니다. 아이폰 뒷면 상단을 단말기에 가까이 대야 합니다. 중앙이나 하단을 대면 인식이 안 됩니다.
2
케이스 분리 후 테스트
금속 케이스나 카드 수납 케이스(교통카드, 신용카드가 들어있는 경우)는 NFC 신호를 차단합니다. 케이스를 완전히 벗기고 다시 시도하세요.
3
강제 재시작
볼륨 올리기 → 볼륨 내리기 → 전원버튼 길게. 일시적인 소프트웨어 오류로 NFC가 비활성화된 경우 재시작으로 해결됩니다.
4
설정에서 NFC 확인
설정 → NFC가 켜져 있는지 확인하세요. (아이폰 7 이상 모델에서 설정에 NFC 항목이 있습니다.) 또한 설정 → 월렛 및 Apple Pay에서 카드가 정상 등록되어 있는지 확인하세요.

## 원인별 — 소프트웨어 vs 하드웨어 판단

자가 해결 가능
NFC 설정 꺼짐 / 케이스 간섭 / 앱 오류
설정에서 NFC를 켜거나, 케이스를 분리하거나, 재시작으로 해결되면 소프트웨어 문제입니다. 월렛 앱에서 카드를 삭제 후 다시 등록하는 것도 도움이 됩니다.
자가 해결 가능
지역 설정 오류 (해외 구매 아이폰)
해외에서 구매한 아이폰은 한국 애플페이가 작동하지 않을 수 있습니다. 설정 → 일반 → 언어 및 지역에서 지역이 '대한민국'으로 되어 있는지 확인하세요.
수리 필요
NFC 안테나 손상 / 단선
낙하 충격이나 수리 과정에서 NFC 안테나 케이블이 손상되거나 연결이 끊어진 경우입니다. 모든 소프트웨어 점검을 해도 NFC가 전혀 반응하지 않으면 하드웨어 수리가 필요합니다.
수리 필요
화면 교체 후 NFC 불량
사설 수리점에서 화면 교체 후 NFC가 안 되는 경우가 있습니다. 화면 교체 시 NFC 관련 케이블을 제대로 연결하지 않았거나 손상시킨 경우입니다.

> **애플페이만 안 되는 경우 vs NFC 전체가 안 되는 경우**: 애플페이만 안 됨 — 월렛 앱의 카드 등록 문제, 은행/카드사 서버 문제, 지역 설정 문제일 가능성이 높습니다. 다른 NFC 기능(NFC 태그 읽기 등)이 정상이면 하드웨어 문제가 아닙니다. NFC 전체가 안 됨 — 애플페이, 교통카드, NFC 태그 읽기 모두 안 된다면 NFC 안테나 하드웨어 고장을 의심해야 합니다.

## NFC 고장 수리는 어떻게 진행되나요?

NFC 안테나는 아이폰 내부 상단에 위치한 얇은 케이블 형태의 부품입니다. 수리는 NFC 안테나 교체 또는 재연결 방식으로 진행됩니다. 화면 교체 후 발생한 NFC 불량의 경우, 케이블 재연결만으로 해결되는 경우도 있습니다.

> **NFC 테스트 방법**: 집에서 NFC 하드웨어가 정상인지 간단히 테스트할 수 있습니다. App Store에서 'NFC 리더' 앱을 다운로드하고, NFC 태그(교통카드, 신용카드 등)를 아이폰 상단에 대보세요. 태그 정보가 읽히면 NFC 하드웨어는 정상이고, 애플페이 문제는 소프트웨어 설정을 확인하시면 됩니다.

> **다올리페어 NFC 수리**: 다올리페어에서는 NFC 불량을 무료로 진단합니다. 소프트웨어 설정 문제라면 현장에서 바로 해결해 드리고, 하드웨어 수리가 필요한 경우 정확한 견적을 안내해 드립니다. 다른 수리점에서 화면 교체 후 NFC가 안 되는 경우도 점검 가능합니다.
