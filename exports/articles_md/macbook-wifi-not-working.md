맥북 와이파이가 안 됩니다. 어제까지 잘 되던 인터넷이 오늘 갑자기 끊기거나, 아예 연결이 안 됩니다. 다른 기기는 잘 되는데 맥북만 안 되는 경우, 대부분 맥북 쪽 문제입니다.

맥북 와이파이 문제는 크게 두 가지입니다. 설정이나 macOS 문제(소프트웨어)이거나, 안테나나 와이파이 모듈 문제(하드웨어)입니다. 지금 해볼 수 있는 것부터 순서대로 정리했습니다.

## 지금 당장 — 이 순서로 해보세요

1
와이파이 껐다 켜기
메뉴 막대 Wi-Fi 아이콘 → 끄기 → 10초 대기 → 다시 켜기. 일시적 연결 오류의 80% 이상이 이것으로 해결됩니다.
2
네트워크 삭제 후 재연결
시스템 설정 → Wi-Fi → 연결된 네트워크 옆 (i) → '이 네트워크 지우기' → 비밀번호 다시 입력해서 연결. 저장된 설정이 꼬인 경우 해결됩니다.
3
NVRAM 초기화
Intel 맥북: 전원 끄기 → 전원 켜면서 Option+Command+P+R 20초 유지. Apple Silicon(M1 이후): 전원을 완전히 끄고 30초 후 다시 켜면 자동으로 초기화됩니다.
4
macOS 업데이트 확인
시스템 설정 → 일반 → 소프트웨어 업데이트. 와이파이 관련 버그 수정이 포함된 업데이트가 있을 수 있습니다.
5
안전 모드로 부팅
전원 끄기 → 전원 켜면서 Shift 키 유지(Intel) 또는 전원 버튼 길게 → 시작 옵션에서 Shift 누르고 안전 모드(Apple Silicon). 안전 모드에서 와이파이가 되면 서드파티 소프트웨어 충돌입니다.

## 원인별 — 소프트웨어 vs 하드웨어 판단

셀프 해결 가능
네트워크 설정 충돌 / macOS 버그
특정 네트워크에서만 안 되거나, macOS 업데이트 후 발생했다면 소프트웨어 문제입니다. 네트워크 삭제 후 재연결, NVRAM 초기화, macOS 재설치로 해결됩니다.
셀프 해결 가능
공유기(라우터) 문제
모든 기기에서 와이파이가 불안정하다면 공유기 문제입니다. 공유기 전원을 뽑고 30초 후 다시 꽂으세요. 5GHz 대역으로 연결하면 안정성이 좋아집니다.
수리 필요
와이파이 안테나 케이블 손상
맥북 와이파이 안테나는 디스플레이 힌지 부분에 있습니다. 힌지 손상이나 낙하 충격으로 안테나 케이블이 끊어지면 신호가 약하거나 아예 안 잡힙니다. 수리점에서 안테나 케이블 교체가 필요합니다.
수리 필요
와이파이 모듈(로직보드) 불량
와이파이 옵션 자체가 사라지거나, 메뉴 막대에 엑스 표시가 뜨면 와이파이 모듈 문제입니다. Apple Silicon 맥북은 와이파이 모듈이 로직보드에 통합되어 있어 메인보드 수리가 필요할 수 있습니다.

> **이런 상황이라면 수리점 방문이 필요합니다**: 와이파이 아이콘에 엑스(X) 표시 — 하드웨어가 와이파이 모듈을 인식하지 못하는 상태입니다. 시스템 설정에서 Wi-Fi 항목 자체가 없음 — 모듈 또는 로직보드 문제입니다. 침수 이력 후 와이파이 불안정 — 내부 부식으로 안테나나 모듈이 손상됐을 수 있습니다.
