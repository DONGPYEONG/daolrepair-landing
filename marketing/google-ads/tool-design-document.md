# 다올리페어 광고 자동화 도구 — 설계 문서

## 1. 도구 개요

**도구명:** 다올리페어 Google Ads 자동화 (Daolrepair Google Ads Automation)

**목적:** 자체 운영 중인 단일 Google Ads 계정의 광고 캠페인, 광고그룹, 키워드, 광고 카피 생성·관리를 자동화하여 운영 효율성 향상.

**사용자:** 다올리페어 매장 운영자 1인 (내부 운영)

**배포:** 외부 공개·판매·공유 없음. 사내 운영 도구.

## 3. 기술 스택

- **언어:** Python 3.11
- **API:** Google Ads API (google-ads Python client library)
- **인증:** OAuth 2.0 (사용자 동의 기반 refresh token)
- **데이터 저장:** 로컬 파일 시스템 (CSV·JSON)
- **실행 환경:** 운영자 개인 PC (macOS)

---

## 4. 주요 기능

### 4.1 캠페인·광고그룹 생성
- 5개 검색 캠페인 (아이폰·아이패드·애플워치·애플펜슬·맥북)
- 캠페인당 4개 광고그룹 (전국·가산·신림·목동)
- 일일 예산, 입찰 전략, 위치·언어 자동 설정

### 4.2 키워드 일괄 등록
- 사내 키워드 풀(약 850개)을 정확 일치·구문 일치 2가지 매치 타입으로 등록
- 키워드별 입찰가 자동 산정 (기존 네이버 운영 데이터 기반)
- 광고그룹별 랜딩 페이지 자동 매칭

### 4.3 반응형 검색 광고(RSA) 생성
- 광고그룹당 헤드라인 15개·설명 4개 자동 등록
- 지점별 텍스트 변형 자동 적용

### 4.4 부정 키워드 관리
- 거래 의도 없는 검색어(무료·DIY·튜토리얼 등) 일괄 차단
- 광고 효율 향상

### 4.5 실적 보고
- 일일·주간 실적 데이터 추출 (노출수·클릭수·전환수·비용)
- 키워드별 ROAS 분석
- CSV 보고서 자동 생성

---

## 5. 데이터 처리 범위

- **처리 데이터:** 자체 광고 계정의 통계 데이터만 (캠페인·키워드 실적)
- **개인정보:** 처리하지 않음
- **외부 공유:** 없음
- **저장 위치:** 운영자 개인 PC 로컬 디스크

---

## 6. API 사용량 예상

- 일일 평균 API 호출: 50~100회 미만
- 주요 호출:
  - 캠페인·키워드 조회 (Get): 일 30~50회
  - 입찰가·예산 조정 (Mutate): 일 10~20회
  - 실적 보고서 추출 (Search): 일 10~30회
- 피크 시 일 호출량: 500회 미만

---

## 7. 보안 및 컴플라이언스

- OAuth 토큰은 로컬 `.env` 디렉토리에 저장, Git 커밋 제외
- 인증 정보 외부 노출 없음
- Google Ads 약관 준수
- 자동화로 인한 정책 위반 행위 없음 (광고 게재 정책·고객 데이터 정책 모두 준수)

---

## 8. 사용 시나리오

1. **초기 캠페인 셋업**: 850개 키워드·20개 광고그룹·5개 캠페인을 한 번에 생성
2. **주간 입찰가 조정**: 효율 좋은 키워드 입찰가 자동 상향, 낭비 키워드 일시중지
3. **신규 키워드 추가**: 시즌별 신규 검색어 발견 시 일괄 추가
4. **광고 카피 A/B 테스트**: RSA 헤드라인 변형 일괄 적용
5. **실적 보고서**: 매주 월요일 자동 생성
