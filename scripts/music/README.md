# Reel BGM 음악 폴더

이 폴더에 **MP3 / M4A / WAV** 파일을 넣어두면, 매일 자동 생성되는 인스타 Reel 영상의 BGM으로 자동 합성됩니다.

## 작동 방식

- 파일이 **0개** → 무음 영상 (인스타에서 직접 음악 선택 가능)
- 파일이 **1개** → 모든 영상에 그 음악
- 파일이 **여러 개** → 영상마다 슬러그 해시로 회전 (같은 일지는 항상 같은 곡)
- **`_` 로 시작하는 파일** → 무시 (보관용)

영상 길이(~17초)에 맞춰 앞부분 자동 자르고, 페이드인 0.6s · 페이드아웃 1.2s · 볼륨 55% 적용.

## 추천 무료 음원 (저작권 안전 · 상업 사용 가능)

브라우저로 아래 사이트에서 검색 → 다운로드 → 이 폴더에 넣기:

### 🎼 1. Pixabay Music (가입 없이 다운, CC0)
https://pixabay.com/music/

추천 카테고리/검색어:
- **"Corporate"** — 정직·전문적
- **"Cinematic"** — 시네마틱 (수리 과정과 잘 어울림)
- **"Tech"** — 미래적·기술적
- **"Inspiring"** — 긍정·자신감
- **"Upbeat Corporate"** — 밝고 빠른 톤

### 🎵 2. Mixkit
https://mixkit.co/free-stock-music/

추천:
- **Corporate Background Music**
- **Cinematic Background**

### 🎶 3. YouTube Audio Library
https://studio.youtube.com → "오디오 보관함"

YouTube에서 다운로드한 음원은 인스타·페북·다른 SNS 모두 OK.

## 권장 BGM 톤 (다올리페어 어울리는 분위기)

| 톤 | 어울리는 영상 |
|---|---|
| **차분·시네마틱** | 후면 유리 / 메인보드 같은 정밀 수리 |
| **밝고 빠른 코퍼레이트** | 배터리 / 충전 같은 빠른 수리 |
| **자신감·인스피레이션** | 침수 처리 / 복구 케이스 |
| **미니멀·미래적** | 신모델(아이폰 15·16 / Apple Watch Ultra) |

**4~6곡 정도 다양한 톤**으로 넣어두시면 — 회전이 자연스럽고 매번 다른 느낌 영상이 만들어집니다.

## 파일명 규칙

특별한 규칙 없습니다. 의미 있는 이름으로:
- `corporate-cinematic.mp3`
- `tech-inspire.mp3`
- `calm-piano.mp3`
- `upbeat-energetic.mp3`

## 무시할 음원

- 가사 있는 노래 (자막 가독성 떨어짐)
- 너무 빠른 BPM (140+ — 영상 흐름과 안 맞음)
- 시작 부분이 조용한 빌드업 (영상이 짧아 fade-in 동안 들리지도 않음)

## 길이

영상이 ~17초이고 앞부분만 잘라 쓰므로, 음원 길이는 **20초 이상**이면 충분합니다. 보통 다운받으면 1~3분짜리라 문제 없음.
